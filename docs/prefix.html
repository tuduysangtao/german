Note on Classification Inconsistencies

Some prefixes appear in both separable and inseparable categories (like “über-” and “an-”). This is because:

  1. Some prefixes can function as either separable or inseparable depending on the verb
  2. When the same prefix + verb combination appears in both categories, they typically have different meanings

For example: - übernehmen (to take over): Usually classified as separable but frequently used as if inseparable - annehmen (to accept): Can be used both ways, with subtle meaning differences
